🤖 cue - Your Private AI Meeting Assistant

Download cue for Windows

cue acts as a silent partner during your digital meetings. This software monitors your audio and video feed to provide real-time notes and actionable insights. It stays invisible to screen sharing applications, so your workspace remains private and professional.

✨ Key Features

  • Smart Transcription: Converts speech to text in real time.
  • Invisible Operation: Does not appear on shared screens during calls.
  • Meeting Summaries: Generates bullet points for your post-meeting review.
  • Privacy First: Uses your own API keys to ensure your data stays under your control.

🚀 Getting Started

Follow these steps to install and run the application on your Windows computer.

1. Visit the Download Page

Go to the official release page to select the latest version for your system.

2. Download the Installer

Look for the file ending in .exe under the Assets section of the latest release. Click the file to download it to your Downloads folder.

3. Run the Setup

Locate the file in your folder and double-click it. Windows may show a security prompt asking for permission to run the software. Select Run to continue.

4. Configure Your API Key

Upon launch, the application asks for an API key. You must provide your own key from your preferred AI service provider. Paste this key into the settings menu to activate the features.

⚙️ System Requirements

Ensure your computer meets these standards for a stable experience:

  • Operating System: Windows 10 or Windows 11.
  • Memory: 8 GB of RAM or higher.
  • Processor: Recent dual-core processor or better.
  • Connection: Stable internet for real-time AI processing.

🔒 Privacy and Data

This software operates locally on your machine whenever possible. You manage your own API credentials. The application does not store your meeting audio on external servers. It only sends necessary text data to the AI service you choose to facilitate the summary and transcription tasks.

💡 Troubleshooting

If the application fails to start, verify that you possess a modern version of Windows. Check your internet connection if the transcription feature shows empty results. Many common issues resolve after you restart the application.
